# Boolean Logic
| X | Value  |
|---|---|
| 0 | false  |
| 1 | true   |

## NOT !
| X | Value |
|---|---|
| !true | false |
| !false | true |

## AND &&
| X | Y  | X && Y |
|---|----|---|
| 0 | 0  | false |
| 0 | 1  | false |
| 1 | 0  | false |
| 1 | 1  | true |

## OR ||
| X | Y | X OR Y |
|---|---|---|
| 0 | 0 | false |
| 0 | 1 | true |
| 1 | 0 | true |
| 1 | 1 | true |
